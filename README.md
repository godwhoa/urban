# A cli tool for urban dictionary 

## Install
Pre-compiled binaries in bin dir.<br>
If you have Go installed and `$GOPATH` set then:
`go get -v github.com/godwhoa/urban`

## Usage
```
# Shows one def.
urban [word] 

# Limits def. to number you specify
urban [word] [limit]

# Shows all def.
urban [word] all
```
